# 🖼️ Screenshots — Violet Bank Security

Esta pasta é destinada às capturas de tela das telas do aplicativo **Violet Bank Security**.

## Como adicionar prints

1. Tire o print da tela desejada no Bubble (modo Preview)
2. Salve o arquivo com o nome correspondente abaixo
3. Faça upload diretamente nesta pasta pelo GitHub (botão `Add file > Upload files`)

## Nomes sugeridos para os arquivos

| Arquivo | Tela |
|---|---|
| `login.png` | Tela de login |
| `signup.png` | Tela de cadastro |
| `dashboard.png` | Dashboard SOC |
| `meus-incidentes.png` | Lista de incidentes do analista |
| `reportar-incidente.png` | Formulário de novo incidente |
| `relatorios.png` | Página de relatórios |
| `incidente-detalhe.png` | Detalhe de um incidente |
| `perfil.png` | Página de perfil do analista |
| `notificacoes.png` | Central de notificações |
| `tarefas.png` | Lista de tarefas |

## Formato recomendado

- **Formato:** `.png` ou `.jpg`
- **Resolução:** Preferencialmente 1280x800 ou maior
- **Modo:** Captura em modo Preview do Bubble (tela cheia)
